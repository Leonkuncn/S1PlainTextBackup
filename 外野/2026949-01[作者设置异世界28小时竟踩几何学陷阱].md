如果 n 是 2 的 k 次方和任意个（可为 0 个）相异费马素数的乘积，那么这个正 n 边形可以用圆规和直尺作出。


这个定理被叫做 Gauss–Wantzel 定理。


已知的费马数中只有前五个是素数：


F0 = 3, F1 = 5, F2 = 17, F3 = 257,和F4 = 65537 （OEIS中的数列A019434）

接下来的二十八个费马数，从F5到F32，已证实都是合数。[1]


因此正n边形如果


n = 3, 4, 5, 6, 8, 10, 12, 15, 16, 17, 20, 24, 30, 32, 34, 40, 48, 51, 60, 64, ... （OEIS中的数列A003401）

则可以用圆规和直尺作出，如果


n = 7, 9, 11, 13, 14, 18, 19, 21, 22, 23, 25, 26, 27, 28, 29, 31, 33, 35, 36, ... （OEIS中的数列A004169）

则不能。
