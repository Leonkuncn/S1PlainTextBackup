【“土猪拱白菜”？衡水学霸真实状况曝光】近日，河北衡水中学17岁高三学生张锡峰“土猪拱白菜”的演讲引发争议。在演讲中，他将自己形容为“一只来自乡下的‘土猪”，来到城市他感受到了落差，其母也曾哭着说“穷人的孩子很辛苦，作为家长很内疚”。而张锡峰读的却是衡水市最贵的私立中学，1年学费够30人上学，还有专属粉丝群…


本文来源：时代周报 作者：陈佳慧
高考结束的第二天，在母校衡水中学的安排下，“土猪”男孩张锡峰坐着一辆三轮车离开了校园，三轮车的车厢里放着他的被褥、课本等，还有一块写着“世界灿烂盛大，欢迎回家”的牌子，学校不远处就是接他回家的父母。
张锡峰的出现，引来了家长的围观，他们纷纷掏出手机，对准上过电视的“明星”学生张锡峰。一位家长一边拍摄视频，一边大声点评：“真好，演讲得真好，说出了咱们农民的心声。”
<img src="https://pics4.baidu.com/feed/e61190ef76c6a7efa7c17c7465d80559f2de66ac.jpeg?token=752358cd2a0a8a0a5b885de1bb82950f" id="aimg_WWHwr" onclick="zoom(this, this.src, 0, 0, 0)" width="599"/)

迎接张锡峰的标语 图/澎湃
张锡峰并未理会，自顾自地把三轮车里的行李搬进轿车里，不一会儿，这辆黑色的帕萨特领驭轿车就被他三年的家当塞得满满当当。张锡峰的父亲开车，母亲坐在副驾，他则和被褥挤在后排。车门关闭，轿车驶离，张锡峰也仿佛人间蒸发一般。
<img src="https://pics3.baidu.com/feed/c83d70cf3bc79f3de19db96d2c836719728b291d.jpeg?token=8fc4536d8f1c99626b017b82f779200f" id="aimg_lRbB2" onclick="zoom(this, this.src, 0, 0, 0)" width="599"/)

张锡峰迈入还贴着保护膜的汽车 图/澎湃
此前十天，17岁的张锡峰因在电视节目《超级演说家·正青春》中演说：“我就是一只来自乡下的土猪，也要立志去拱了大城市里的白菜”而备受争议。6月15日下午，衡水中学就张锡峰所受争议一事回复时代周报记者称：“我这没有接到相关的材料，无法回复你这个问题。”
初高中花费近20万
在《超级演说家·正青春》的节目中，张锡峰作了题为“小小的世界**的你”的演说。正是这10分钟的演讲，使得这位将要成年的高三学生饱受争议。
在演讲中，张锡峰说他出生于河北省东部的一个小小乡村，“小镇不大，但傍晚的风总是很温柔”。衡水的风温不温柔未可知，但衡水冬天气温低至零下10℃并不稀奇。
在张锡峰的演讲中，他说在那个小小乡村，他度过了平平淡淡将近10年的时光。从一所只有两间屋子、三个年级的乡村小学来到了城里读书。
<img src="https://pics3.baidu.com/feed/c8177f3e6709c93df2fa39430f1f52d4d1005420.jpeg?token=fbe18f577b99e005a9e5b8e6a18c044f" id="aimg_Ijj5J" onclick="zoom(this, this.src, 0, 0, 0)" width="599"/)

从农村来到了城市，张锡峰感受到了落差：“我用家乡话做出的自我介绍惹得全班同学哄堂大笑”“因分不清自家球门和对手球门，而被所有队友指责“放学后同学被家长用轿车接走，而我只能背着沉重的书包一步一步走路回家”。
他在演讲中将自己形容为“一只来自乡下的‘土猪”。而这只“土猪”却转头进了衡水市最贵的私立中学。
衡水市第五中学的相关负责人对时代周报记者表示，初中三年，张锡峰确是在该校就读。衡水市第五中学又名衡水桃城中学，是一所创办于1978年的民办中学，“我们学校约有2万名学生，全部都是寄宿，升学率已经是我们区‘十二连冠’了”。
<img src="https://pics1.baidu.com/feed/35a85edf8db1cb13b11e6c054576fc4692584b58.jpeg?token=4a7ead70c16e7df9373b1dfd99a44cb0" id="aimg_q6944" onclick="zoom(this, this.src, 0, 0, 0)" width="599"/)

衡水市桃城中学校区
在衡水桃城中学一年要花费多少钱？上述工作人员透露：“桃城中学一年学费3万元，住宿费2000元，吃饭是吃多少花多少。”如果仅算学费和住宿费，初中三年，张锡峰的父母就需要支付9.6万元的费用。
若按照一年34周教学时间，一周6天，一天花费30元伙食费计算，张锡峰一年的伙食费用最少需要6120元。即不算其他开支，初中三年，张锡峰上学的费用已超10万元。
根据衡水市物价局、衡水市教育局《关于调整全市公办普通高中学费标准的通知》，衡水市市区省级示范性高中每生每学期700元，一般高中每生每学期500元。仅学费一项，张锡峰一年的学费就够30个学生上一般高中一年。
据衡水市统计局发布的《衡水市2017年国民经济和社会发展统计公报》显示，2017年，张锡峰上初二时，衡水市居民人均可支配收入18004元。其中，城镇居民人均可支配收入26195元，农村居民人均可支配收入11194元。张锡峰初中三年的学费，相当于9个农民一年的可支配收入。
<img src="https://pics0.baidu.com/feed/c75c10385343fbf2c97e3592285c608864388fd8.jpeg?token=fa2f7d974e859dfe07c8ae0045dab7f7" id="aimg_zy23Y" onclick="zoom(this, this.src, 0, 0, 0)" width="599"/)

衡水中学高考誓师大会
据衡水市一位从事教育咨询的工作人员介绍，张锡峰就读的高中——衡水中学，一年学费1.8万元，住宿费2000元，吃饭的钱另算。
若仍以初中标准计算伙食费，初中、高中6年，张锡峰至少花费192720元，平均一年花费32120元。
据衡水市统计局发布的《衡水市2020年国民经济和社会发展统计公报》显示，2020年，张锡峰上高二时，衡水市居民人均可支配收入23527元。其中，城镇居民人均可支配收入33223元，农村居民人均可支配收入15100元。
在上述演讲节目中，张锡峰母亲陈照莲哭着说：“穷人的孩子很辛苦，我觉得没有给孩子一个比别人特别好的环境，我和他爸爸我们俩做不到，我们只是农民，普通的农民，对孩子来说，我感到很内疚。”
若张锡峰真如母亲所说是“穷人的孩子”、“农民的孩子”，那花费20多万元供孩子读书，父母真可谓“砸锅卖铁”了。按照统计公报数据计算，如果张锡峰父母是平均收入水平的农民，那么两个人不吃不喝，刚好够张锡峰一年的花费。
微博超话里的4719个“西风女孩”
饱受争议的不仅是张锡峰的“一只来自乡下的‘土猪’”人设，更是他“也要立志去拱了大城市里的白菜”的戾气。
<img src="https://pics1.baidu.com/feed/faedab64034f78f08c9c9512ee13a05db2191c62.jpeg?token=42a8c0dc43f754518cb7ab6e4be317b4" id="aimg_SbfXr" onclick="zoom(this, this.src, 0, 0, 0)" width="599"/)

张锡峰的微博超话
作为衡中十大年度人物，学校里挂着张锡峰双手抱胸的照片，照片下面配文：“他热爱演讲，热爱身边的人。在高二最关键而紧张的一年，他身兼数职——班长、课代表、纪检部成员。”
张锡峰确实爱演讲，高一学期末在学校做“青春与梦想”主题演讲；高二时，给初中母校的学弟学妹演讲；高三时，参加央视节目《中国少年说》、安徽卫视《超级演说家·正青春》。
他的演讲视频里，“偶像”“男神”“学霸”“他在发光”的弹幕不断弹出。衡水中学的贴吧里有不少倾慕者在求张锡峰的QQ号，抑或退一步求他的网易云账号或者喜马拉雅的账号。
<img src="https://pics6.baidu.com/feed/34fae6cd7b899e51ec93c92cd085733bc8950d90.jpeg?token=00bf4bfc9cfee035b12ea5625f48dcf9" id="aimg_Ci23s" onclick="zoom(this, this.src, 0, 0, 0)" width="599"/)

张锡峰的喜马拉雅也有2000多名粉丝
在喜马拉雅平台上，一个名为“张嘻疯”的账号里有6条演讲音频，题为“要么庸俗，要么考试”“等待失落的灵魂”等，声音确系张锡峰。而疑似张锡峰创建的网易云歌单里，一个是100多首纯音乐，另一个则是某网络歌手的歌曲合集。歌单里的评论多是“高考加油”“你是大家的信仰啊！”“你是我的光啊”。
在微博平台，张锡峰已拥有了自己的超话，而且“衡水中学张锡峰”超话里则聚集了4719个“西风女孩”。“西风女孩”为“锡峰女孩”的谐音，也是张锡峰粉丝的自称。
<img src="https://pics7.baidu.com/feed/94cad1c8a786c91702f59b7f511fdac73ac757a9.jpeg?token=4f637ee9333b501faf412ef1d193c42c" id="aimg_Ha4hT" onclick="zoom(this, this.src, 0, 0, 0)" width="599"/)

“西风女孩”在超话中为张锡峰打气
“西风女孩”们在超话里为他辩解：“他们不懂张锡峰的所有演讲给中考、高考生带来了多大的动力。”“一个励志演讲居然能被骂成这样，世风日下啊”“真的希望张锡峰不要看到网上那些负面评价，我的白月光啊！”“西风女孩不认输！”

其实，早在2019年，即张锡峰读高一时，就有了追随者“西风女孩”，“我看了他的演讲后爱上了他”，另一位女生在下面评论“我也是”。

“西风女孩”在超话中的发言
一条发布于2019年的微博写道：“用一个词形容你眼中的张锡峰”。三年里陆陆续续有人评论，在张锡峰读高一时，评论里写着“光芒万丈”“宝藏男孩”“我的男孩”“人间理想”；他读高二时，评论又增加了“白月光”“我看过的那些小说男主突然有了清晰的模样”“无法企及的高度，但我会努力踮起脚尖靠近你”；张锡峰读高三时，评论的内容则是“他站在那里就是别人的风景”“他就是心中的万物”。
