古力特补完了，大概观后感↓


(+)两位女主的人设


(+)校园(JK)感的画面，以及非常有趣的语音小剧场


(/)中规中矩的特摄


(-)用了很多笔墨塑造悬疑感，但由刻意回避基础设定


本质上是一个特摄片+校园(JK)片，里面充满了大量的特摄经典桥段，如果对特摄不感兴趣恐怕对这个片兴趣也不大。


特摄方面，中规中矩，手绘虽然有一种独特画风，但是古力特和怪设都没有很帅或者很潮的感觉。


人物方面，很多细节塑造的JK感非常非常强烈，最直接的例子就是ED里的各种画面。但是到剧情中后期，工具人感就开始显现了。我理解可能有以下原因：所有人物都是新条茜的造物，设定上就不是真正意义上的人类；每一个角色既没有基底（某种残缺等）也几乎没有角色弧；篇幅原因。当然工具人是特摄的**病，把SSSS当特摄片看也能接受这种程度的工具化。不少地方能看出致敬EVA的痕迹，制作班底也有一些联系，但是EVA就是EVA，古力特和EVA的差距真是在天在地。


日常Part和非日常Part的区别并没有之前评论说的那么大，个人观感里也没有电光机王第一话那么明显，究其原因，是因为整个日常Part就充满了怪兽、失忆、记忆重置等等非日常要素。此外，更多单纯表现日常也就是人物的部分则被丢进了声音小剧场里。拜托了画面的限制，声优语速更快更贴近日常，台本信息量也很大，最后做出的这个小剧场就特别有意思，建议看过正片的一定要看。


一个最大的缺憾是世界观的设定基本没有说清楚，哪怕作为特摄片，基本的设定我觉得还是要说清楚的。EVA展示了很多，并且潮，没展示的部分就会吸引人；SSSS没展示多少，展示的部分也不够潮，这时候没讲清楚基本设定只会导致两种结果：观众无所谓或者观众扣分。


我是不支持第三新东京市和EVA的故事都是真嗣的幻想的，因为那样解构了整个故事；但是古力特里则是明明确确指出了整个世界都是新条茜的造物。眼镜上的裂痕和内心的崩坏始终存在。这一方面会让我觉得整个世界包括主角团都是新条茜内心的映射（从这个角度可以很容易说清楚裕太的特殊性，裕太是大家都围着新条茜这一被包装过的神明时，唯一看向六花的，而六花则是新条茜内心某处的一种未包装过的映射），自然也就不存在世界系的味道。但是另一方面，新条茜仍然独立于这个世界，关键的被拯救要依靠和自己有直接关系的造物，安奇以及六花来达成，而不能直接依靠这个世界的运作。这方面讲又有一些世界系的味道。比较模糊，我也不想深究。概括来看，剧情和情绪的表达不是很戳我，比如我能体会到新条茜一直在崩坏，却比较难体会到捅刀之前新条茜的“绝望感”，还有像“裕太其实是古力特”这个“反转”我也相当无感。


总的来说：特摄方面中规中矩，手绘提供了另一种风味；校园感和JK感由一个个小场景体现，更多的精髓放在了语音小剧场里；剧情和人物本身塑造的一般般。

转自朋友的分析：


想谈一谈SSSS.DYNAZENON中关于雨伞的意象。
伞是用来遮挡雨水，以防雨水浸湿自己，导致自己感冒生病。个人认为这种功能在稍加转换以后，可以变成符合本作主题的理解：雨伞是人与人之间的隔阂。<img src="https://pic1.zhimg.com/v2-4a55c2ea792b89bb6b6569d24ad80772_720w.jpg?source=3af55fa1" height="409" id="aimg_uo3X3" onclick="zoom(this, this.src, 0, 0, 0)" width="720"/)在第一集中，梦芽向未曾搭过话的蓬邀约，想让他今晚九点陪自己。这时，她并没有带着伞。<img src="https://pic3.zhimg.com/v2-2dea012f322bd054881bef8cdfd88977_720w.jpg?source=3af55fa1" height="411" id="aimg_S7qZ7" onclick="zoom(this, this.src, 0, 0, 0)" width="720"/)然而到了晚上，她的身旁却多了一把伞。<img src="https://pic2.zhimg.com/v2-d61d820ff7e6a9f1da28b97c1f400542_720w.jpg?source=3af55fa1" height="411" id="aimg_GEbbB" onclick="zoom(this, this.src, 0, 0, 0)" width="720"/)<img src="https://pic1.zhimg.com/v2-fb24b16aaa22e95e7bdf7b7fd0f293c7_720w.jpg?source=3af55fa1" height="408" id="aimg_G766S" onclick="zoom(this, this.src, 0, 0, 0)" width="720"/)望着远处淋雨的蓬，梦芽不仅没有感到内疚或心疼，反而是拉下雨伞，继续观察蓬。<img src="https://pic2.zhimg.com/v2-690514eb49e8acc23a5fa34597f924b9_720w.jpg?source=3af55fa1" height="408" id="aimg_m0E0e" onclick="zoom(this, this.src, 0, 0, 0)" width="720"/)初战结束后，梦芽的伞不见了。

将这几段剧情和雨伞串起来的个人理解便是：

梦芽在一开始约蓬的时候，或许内心真的期望自己能和他见上一面，敞开心扉地说出自己内心困恼许久的话题。然而，真正等到赴约时，她虽然按时赶到，却只是在远处看着蓬。

所谓的雨伞便是她用来隔绝自己与外界交流的壁垒。

雨伞的出现意味着她放弃了与他人的沟通交流，雨伞下拉的镜头表达出她内心这种矛盾的情绪被强化。

可是，在怪兽这种“非日常”的东西出现后，被强制塞到一起的少年少女们不得不“合体”迎战，她心中的壁垒以一种暴力的方式崩坏，怪兽和DYNAZENON的存在使得她不得不和其他人进行交流。

雨伞的消失，也意味着她闭塞的心房逐渐打开。

<img src="https://pic1.zhimg.com/v2-e1e0205563dde623f6ac07381f37f832_720w.jpg?source=3af55fa1" height="406" id="aimg_X1hwx" onclick="zoom(this, this.src, 0, 0, 0)" width="720"/)

那么，之前在第二集中大量雨伞被破坏的画面也能得到合理的解释：这个世界中闭塞的人心因怪兽和DYNAZENON间产生的激斗被强制打开，阻隔沟通交流的壁垒也因此崩塌。人和人之间因一种不情愿的外力作用而产生联系。虽然这个外力作用本身是坏的，但带来的效果却不一定是坏事。这也许就是失马名字的合理解释。

顺带提一句，历和千濑同样也带着雨伞。不过，历带着的是黑色的雨伞，其隔阂程度更深一些——无法通过雨伞看到外界，外界也无法看到自己，象征着自己与外界联系完全被切断。这与他的人设也有些许符合。

淋雨的蓬和失马则更不必多说。他们虽然不会主动说出自己的心事，但他们是乐于倾听的人，至少在交流这一层面而言，他们是一个积极向上的态度。
